# 工作周报助手 安装说明

> 番茄钟驱动的周报生成桌面工具。本说明面向 macOS（Apple Silicon：M1 / M2 / M3 / M4 芯片）。

## 一、安装步骤

### 1. 打开安装包

双击收到的文件 **`工作周报助手_0.1.0_aarch64.dmg`**。

会弹出一个安装窗口，左边是 **工作周报助手.app** 图标，右边是蓝色的 **Applications（应用程序）** 文件夹。

### 2. 拖拽安装

把左边的 **工作周报助手.app** 图标，拖到右边的 **Applications** 文件夹上，松手。

完成后可以弹出（推出）这个 dmg 窗口，dmg 文件本身之后可以删除。

### 3. 首次打开

打开「启动台」或「应用程序」文件夹，找到 **工作周报助手**，双击运行。

> 💡 如果双击后弹出提示「**已损坏，无法打开**」或「**无法验证开发者**」，请先按下面「二、首次打开报错怎么办」处理，再回来双击运行。这是 macOS 对未签名应用的正常拦截，**不是文件坏了**。

---

## 二、首次打开报错怎么办

如果双击时提示「**已损坏，无法打开**」或「**无法验证开发者是否包含恶意软件**」，按下面任选一种方式处理即可：

### 方式 A：终端命令（推荐，一次解决）

1. 按 `Command + 空格` 打开「聚焦搜索」，输入 **终端** 或 **Terminal**，回车打开「终端」App。
2. 复制下面这一行，粘贴到终端窗口里，按回车：

   ```
   xattr -cr /Applications/工作周报助手.app
   ```

3. 没有任何输出就是成功了（正常现象）。回到「应用程序」文件夹，重新双击 **工作周报助手** 即可运行。

### 方式 B：系统设置放行（不习惯命令行可用这个）

1. 双击 **工作周报助手**（即使提示已损坏也先双击一下）。
2. 打开「系统设置」→「**隐私与安全性**」。
3. 滚动到页面下方，会看到一条提示：「**"工作周报助手"已阻止打开**」，旁边有「**仍要打开**」按钮，点击它。
4. 按提示输入开机密码确认，再次双击应用即可运行。

> 两种方式二选一即可。方式 A 一劳永逸，方式 B 更直观。

## 三、常见问题

**Q：提示「已损坏」是不是文件传坏了？**
不是。这是 macOS 的隔离机制——通过聊天工具（微信 / 钉钉 / 邮件 / 网盘）收到的文件会被打上隔离标记，本地自制的应用没有 Apple 官方签名就会触发这个提示。按上面「二」处理一次就好。

**Q：以后每次更新都要重新执行命令吗？**
如果应用安装位置和文件名没变，不用。`xattr -cr` 清一次即可。换新版本覆盖安装后如果又提示，再执行一次同一命令即可。

**Q：数据存在哪？会丢吗？**
你的周报数据存在 `~/Library/Application Support/com.remix.weeklyreport/weekly.db`，与应用本体分开。卸载应用不会丢数据，但**建议定期备份**这个文件。

**Q：不是 Apple Silicon 芯片能用吗？**
本版本仅适配 Apple Silicon（M 系列芯片）。Intel 芯片的 Mac 无法运行此版本。

---

## 四、卸载

1. 把「应用程序」里的 **工作周报助手.app** 拖到废纸篓，清空废纸篓。
2. 如需同时清除数据，删除这个文件即可（会清空所有周报记录，谨慎）：
   ```
   ~/Library/Application Support/com.remix.weeklyreport/weekly.db
   ```
